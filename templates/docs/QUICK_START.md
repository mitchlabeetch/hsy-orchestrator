# HSY Quick Start Guide - v3.0

## 🚀 Start in 5 Seconds

```bash
hsy-launch
```

That's it! One command launches:
- ✅ Self-tests
- ✅ Beautiful TUI v3.0
- ✅ AI-powered orchestrator
- ✅ Fully autonomous execution

---

## ⌨️ TUI v3 Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `S` | Start/Stop orchestrator |
| `P` | Pause/Resume execution |
| `M` | Model routing settings |
| `H` | Execution history |
| `?` | User questions |
| `R` | Refresh display |
| `Q` | Quit |

---

## 🤖 Model Presets

Press `M` to choose:

1. **Ultimate** - Sonnet 4.5 (best quality)
2. **Balanced** - Sonnet 4 (default, recommended)
3. **Credit-Saver** - Sonnet 3.5 + Haiku (efficient)
4. **Custom** - Choose per role

---

## 🎯 Workflow

1. Launch: `hsy-launch`
2. System auto-starts
3. Detects scenario
4. Executes hooks automatically
5. Pauses for questions if needed
6. Continues until complete

**No manual intervention needed!**
